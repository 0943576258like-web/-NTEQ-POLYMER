# NTEQ POLYMER — PHP CMS

ระบบจัดการเนื้อหาแบบสำเร็จรูป ใช้ PHP + SQLite ไม่ต้องตั้งค่าฐานข้อมูล

## วิธีติดตั้ง

1. อัปโหลดทุกไฟล์ขึ้นโฮสต์ PHP (ต้องการ PHP 7.4+ และเปิด PDO_SQLite)
2. ตั้งสิทธิ์ให้โฟลเดอร์ `data/` และ `uploads/` เขียนได้ (chmod 775)
3. เปิดเว็บเบราว์เซอร์ที่ `https://โดเมนของคุณ/` → ฐานข้อมูลจะสร้างอัตโนมัติ
4. เข้าหลังบ้านที่ `/admin/` ด้วย:
   - **อีเมล:** `admin@nteq.local`
   - **รหัสผ่าน:** `admin1234`
5. **เปลี่ยนรหัสผ่านทันที** ในเมนู "บัญชีผู้ดูแล"

## โครงสร้าง

```
index.php          หน้าแรก
page.php           หน้าเมนูต่างๆ (?slug=about)
article.php       หน้าบทความ (?slug=...)
admin/             หลังบ้าน
  login.php
  index.php        แดชบอร์ด
  hero.php         แก้ Hero / เกี่ยวกับเรา
  pages.php        จัดการเมนู
  articles.php     จัดการบทความ
  contact.php      ข้อมูลติดต่อ
  account.php      บัญชี / เปลี่ยนรหัส
data/cms.sqlite    ฐานข้อมูล (ห้ามลบ)
uploads/           รูปภาพที่อัปโหลด
```

## ฟีเจอร์

- ✅ ล็อกอินด้วยอีเมล + รหัสผ่าน (bcrypt + CSRF)
- ✅ แก้ไขข้อความหน้าแรก / Hero / CTA / รูปพื้นหลัง
- ✅ เพิ่ม/แก้/ลบ หน้าเมนูได้ไม่จำกัด (รองรับ HTML)
- ✅ เพิ่ม/แก้/ลบ บทความ พร้อมรูปหน้าปก ปุ่ม READ MORE อัตโนมัติ
- ✅ แก้ข้อมูลติดต่อ + ฝัง Google Maps
- ✅ อัปโหลดรูป (jpg/png/webp/gif สูงสุด 8MB)
- ✅ ป้องกันการเข้าถึงฐานข้อมูลและการรันโค้ดในโฟลเดอร์อัปโหลด

## หมายเหตุ

- ใช้ Apache + mod_rewrite จะรองรับ `.htaccess` ที่ป้องกันให้แล้ว
- ถ้าใช้ Nginx ต้องตั้ง `deny all` ใน config สำหรับ `data/` และห้าม execute PHP ใน `uploads/`
